State: 사물이나 사람의 물리적, 정신적 상태  
Status: 경과되는 상태 중 한 장면이나 특정 상황에 사용됨  

즉 server의 상태는 'running', 'stopped' 등의 상태를 가지므로 'status'  
현재 server의 상태를 말할 때는 'running state'와 같이 사용된다.
